### Concepts
* Domain driven design (DDD)
    * Domain Events
    * Anti-corruption layer
    * Aggregates
    * Repositories
    * Services
    * Value Objects
* Test-driven development (TDD)
    * Unit tests
    * Integrations tests
* SOLID Concepts
* Inversion of control principles (IoC)
* Unit of work (WOW)
* Message BUS
